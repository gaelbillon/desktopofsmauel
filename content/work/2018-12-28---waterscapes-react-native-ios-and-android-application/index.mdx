Waterscapes is a mobile application for iOS and Android created using React Native. This application is the mobile application version of the static website [waterscap.es](/work/waterscap-es-un-site-statique-realise-avec-gatsby/) made with [Gatsby.js](/gatsby-le-generateur-de-site-statique-base-sur-react-et-graphql).

The data are stored on the [headless CMS](/cms-headless-avantages-inconvenients-comparatif-des-5-leaders) (Contentful) and are retrieved with the API.

### Frameworks and tools :
- React Native
- Visual Studio
- Xcode
- CMS Headless
- Node.js
- Xcode
- Android Studio

### Node modules :
- react-native-image-picker
- react-native-maps
- react-native-markdown-view
- react-navigation
- redux
